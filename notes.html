A bit about the internet

URL

URL stands for Uniform Resource Locator. It contains several different parts, including:
  • The protocol: e.g. http, https, ftp
  • The host: e.g.www.facebook.com,www.bbc.co.uk
  • The path: e.g. /news/local, /john.smith.95
## DNS DNS stands for Domain Name System. You buy a domain name from a registrar (e.g. 123-reg, GoDaddy). They arrange for your server's IP address to be added next to the domain name on a DNS server, so that people can find your site.
